# Portfolio Website

## Overview

This repository contains the code for a personal portfolio website built using HTML, CSS, and JavaScript. The website showcases personal projects, skills, and contact information. It is designed to be deployed to AWS S3 for hosting.

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Deployment**: AWS S3

## Prerequisites

Before you begin, ensure you have met the following requirements:

- AWS account with access to S3
- Basic knowledge of HTML, CSS, and JavaScript

## Project Structure

- `index.html`: The main HTML file for the portfolio.
- `styles.css`: The main CSS file for styling the portfolio.
- `script.js`: The JavaScript file for interactivity and additional functionality.
- `images/`: Directory containing image assets used in the portfolio.
